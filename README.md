## Tech stack

- **React** + **TypeScript**
- **Vite**
- **Tailwind CSS** + **shadcn/ui**
- **React Router**
- **Framer Motion**

## Getting started

```bash
npm install
npm run dev
```

Build + preview locally:

```bash
npm run build
npm run preview
```

## Deployment

This is a standard Vite static site—deploy it anywhere that can host static assets:

- **Vercel / Netlify / Cloudflare Pages**: connect the repo, set build command to `npm run build`, publish directory to `dist`.

## Notes

- The site is designed to be readable and scannable for hiring managers: clear role headline, proof via shipped products, and direct contact paths.
